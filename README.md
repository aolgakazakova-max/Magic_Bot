# Magic_Bot
# 🤖 Telegram AI Bot (ChatGPT + Excel Analysis)

Телеграм-бот с интеграцией ChatGPT и анализом Excel-файлов.

## 🚀 Возможности

- 💬 Диалог с ChatGPT
- 🎲 Генерация случайных фактов
- 🧠 Диалог с историческими личностями (Ван Гог, Гагарин, Авиценна)
- 📊 Анализ Excel-файлов с помощью AI
- 🔄 Перевод текста
- ⏱️ Викторина

---

## 🛠️ Технологии

- Python 3.10+
- aiogram
- OpenAI API
- pandas
- matplotlib
- seaborn
